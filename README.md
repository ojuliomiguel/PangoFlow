# PangoFlow

Initial ClojureScript shell for the PangoFlow browser extension.

## Development

```bash
npm install
npm run dev
```

Load the `public/` directory as an unpacked extension in Chrome or another Chromium browser.

During development, the static pages are also available at:

- `http://localhost:8020/popup.html`
- `http://localhost:8020/dashboard.html`
